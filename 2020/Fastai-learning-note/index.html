Things that can go wrong

LR too high

valid_loss > 1

Cannot undo, just re-creat the model

LR too small

learning really slow, train_loss > valid_loss

Too few epochs

train_loss > valid_loss

Too many epochs

Overfitting

Vocab

  • Learning Rate - The rate of learning (The thing we times our gradient every time)
  • Epoch - One complete run of our datas
  • Minibatch - A random set of our data
  • Model / Architecture - A mathematical function that we need to find the parameters
  • Parameters - Coefficient, the number that we are updating in training
